# AccountManager

Простое Windows Forms приложение для управления учетными записями пользователей.

## Описание

Приложение позволяет:

* Просматривать список пользователей
* Добавлять новых пользователей
* Редактировать существующих
* Удалять пользователей

Данные пользователей хранятся в XML-файле.

## Технологии

* C#
* Windows Forms
* .NET Framework 4.7.2

## Структура проекта

* `Forms/` — формы интерфейса (MainForm, UserEditForm и др.)
* `Models/` — модели данных (User)
* `Services/` — работа с данными (сохранение/загрузка XML)
* `Program.cs` — точка входа в приложение

## Запуск

1. Открыть решение `AccountManager.slnx` в Visual Studio
2. Собрать проект
3. Запустить приложение

## Требования

* .NET Framework 4.7.2
* Visual Studio (рекомендуется 2019 или новее)

## Назначение

Учебный проект для демонстрации работы с Windows Forms и базовой CRUD-логики.
